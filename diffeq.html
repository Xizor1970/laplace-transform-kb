The Three-Step Method

Every ODE follows this recipe

Step 1 — Transform: Take the Laplace Transform of both sides. Derivatives become polynomials in s. Initial conditions appear automatically.

Step 2 — Solve algebraically: The ODE is now an algebraic equation in Y(s). Solve for Y(s) using standard algebra.

Step 3 — Invert: Find y(t) = ℒ⁻¹{Y(s)} using partial fractions and the transform table.

That's it. No guessing particular solutions, no variation of parameters, no undetermined coefficients. Just algebra.

The three-step method is like using logarithms to multiply: take the log (transform), add (do algebra), take the antilog (invert). Just as logs convert multiplication to addition, the Laplace Transform converts differentiation to multiplication. The "hard" operation becomes the "easy" operation in the transformed domain.

Example 1: First-Order ODE

Problem: y' + 3y = 6, y(0) = 2

Step 1 — Transform both sides:

[sY(s) − y(0)] + 3Y(s) = 6/s
[sY(s) − 2] + 3Y(s) = 6/s

Step 2 — Solve for Y(s):

Y(s)(s + 3) = 6/s + 2
Y(s) = 6/[s(s+3)] + 2/(s+3)

Partial fractions: 6/[s(s+3)] = 2/s − 2/(s+3)

Y(s) = 2/s − 2/(s+3) + 2/(s+3) = 2/s

Step 3 — Invert:

y(t) = 2

Check: y' = 0, so y' + 3y = 0 + 6 = 6 ✓, and y(0) = 2 ✓. The initial condition happens to already be the steady state!

Example 2: Second-Order — Damped Oscillator

Problem: y'' + 4y' + 13y = 0, y(0) = 1, y'(0) = 0

This is a damped spring-mass system released from displacement 1 with zero velocity.

Step 1 — Transform:

[s²Y − s·1 − 0] + 4[sY − 1] + 13Y = 0
s²Y − s + 4sY − 4 + 13Y = 0

Step 2 — Solve:

Y(s)(s² + 4s + 13) = s + 4
Y(s) = (s + 4)/(s² + 4s + 13)

Complete the square: s² + 4s + 13 = (s+2)² + 9 = (s+2)² + 3²

Rewrite numerator: s + 4 = (s+2) + 2

Y(s) = (s+2)/[(s+2)²+9] + (2/3)·3/[(s+2)²+9]

Step 3 — Invert from table:

y(t) = e−2tcos(3t) + (2/3)e−2tsin(3t)

The system oscillates at 3 rad/s while decaying exponentially at rate 2. The oscillation dies out with time constant 1/2 = 0.5 seconds.

This solution describes a guitar string plucked and released — it vibrates (the sin and cos terms at frequency 3) while the amplitude fades (the e−2t envelope). The decay rate (2) and frequency (3) came directly from the polynomial s² + 4s + 13 — its roots −2 ± 3j encode both the frequency and the damping in their real and imaginary parts.

Example 3: Forced System with Discontinuous Input

Problem: y'' + y = u(t−π), y(0) = 0, y'(0) = 1

A harmonic oscillator hit by a unit step at t = π. Classical methods struggle with the discontinuity — Laplace handles it effortlessly.

Step 1 — Transform:

[s²Y − 0 − 1] + Y = e−πs/s

Step 2 — Solve:

Y(s) = 1/(s²+1) + e−πs/[s(s²+1)]

Partial fractions for the second term: 1/[s(s²+1)] = 1/s − s/(s²+1)

Step 3 — Invert (using time-shift property for e−πs terms):

y(t) = sin(t) + u(t−π)·[1 − cos(t−π)]

For t < π: y = sin(t) (free oscillation from initial velocity). For t ≥ π: the step force pushes the oscillator to a new equilibrium at y = 1, around which it oscillates.

Systems of ODEs

Multiple equations at once

For a system of n coupled ODEs, the Laplace Transform converts each equation into an algebraic equation in the n unknowns Y₁(s), ..., Yₙ(s). You solve the resulting linear system (Cramer's rule, elimination, or matrix inversion) and invert each component separately.

Example: Two coupled tanks:

x' = −3x + y, x(0) = 2
y' = x − 3y, y(0) = 0

Transform: sX − 2 = −3X + Y and sY = X − 3Y

Solve: (s+3)X − Y = 2 and −X + (s+3)Y = 0

From the second equation: X = (s+3)Y. Substitute: (s+3)²Y − Y = 2, giving Y = 2/[(s+3)²−1] = 2/[(s+2)(s+4)]

Partial fractions and inversion gives:

x(t) = e−2t + e−4t
y(t) = e−2t − e−4t

Integro-Differential Equations

Equations with both derivatives and integrals

Some physical systems (particularly electrical circuits) naturally produce equations with both derivatives and integrals of the unknown. Classical methods require differentiating the whole equation to remove the integral — adding an order and requiring extra initial conditions.

Laplace handles both simultaneously: derivatives give multiplication by s, integrals give division by s. No extra manipulation needed.

y'(t) + 2y(t) + 3∫₀ᵗ y(τ)dτ = e^(−t), y(0) = 1 An integro-differential equation — straightforward with Laplace

Solution

Transform: [sY − 1] + 2Y + 3Y/s = 1/(s+1)

Multiply through by s: s²Y − s + 2sY + 3Y = s/(s+1)

Y(s²+2s+3) = s + s/(s+1) = s(s+2)/(s+1)

Y = s(s+2)/[(s+1)(s²+2s+3)] = s(s+2)/[(s+1)((s+1)²+2)]

Partial fractions and completing the square yields the damped oscillatory solution.

Frontiers

  • Can the Laplace method be extended to nonlinear ODEs in any systematic way, or is linearity fundamentally required?
  • For fractional-order differential equations (with dα/dtα where α is not integer), does the Laplace method generalise cleanly?
  • Are there classes of PDEs where a "Laplace-like" transform in one variable reduces them to ODEs that can be solved systematically?

The Laplace Transform as a universal solver architecture

The three-step method (transform → algebra → invert) is actually an instance of a much deeper pattern in mathematics: solve hard problems by mapping them to an easier domain. Logarithms map multiplication to addition. The Fourier Transform maps convolution to multiplication. Group representations map abstract algebra to linear algebra. In each case, the "transform" finds a domain where the operation of interest becomes trivial.

The Laplace Transform's specific genius is mapping differentiation (which is hard — it introduces initial conditions, requires smoothness, compounds errors) to multiplication by s (which is trivial). The cost is inversion — getting back from s-domain. But since the inversion is routine (partial fractions), the trade is enormously favourable.

This "transform to easy domain" pattern works spectacularly for linear problems. For genuinely nonlinear ODEs (where the superposition principle fails), no comparable universal simplification is known — this is arguably why nonlinear dynamics remains fundamentally harder.
