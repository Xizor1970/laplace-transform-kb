## Features

- **Dark theme** with gold/blue accent colour system
- **Interactive Canvas animations** — drag sliders, see dynamics change in real-time
- **Analogies on every page** — every abstract concept connected to something tangible
- **Speculation boxes** — deeper insights and open questions at the frontier
- **Zero dependencies** — pure HTML, CSS, and vanilla JavaScript
- **Fully static** — opens directly in any browser, no build step
